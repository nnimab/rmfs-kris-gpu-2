1. Population Pg

A population of N neural network individuals {θ1, θ2, ..., θN}
↓

2. Evaluation

Each individual θi is deployed in a separate simulation.
Calculate a fitness score F(θi) based on global performance.
↓

3. Selection

Select the best-performing individuals (Parents) based on fitness scores.
(e.g., using Tournament Selection)
↓

4. Reproduction (to create Pg+1)

Elitism
(Copy best individuals)
Crossover
(Mix parent genes)
Mutation
(Add random noise)
↺

Loop back to New Population Pg+1
